# Tasks

Fonte oficial do backlog: esta pasta.

## Estrutura de status
- `TODO/`
- `DOING/`
- `REVIEW/`
- `DONE/`
- `BLOCKED/`

## Convencoes
- ID: `TASK-YYYY-NNN`
- 1 arquivo por task
- Nome sugerido: `TASK-YYYY-NNN - titulo-curto.md`

## Regra principal
Nenhum agente deve executar trabalho sem `task_id` explicito e existente.

## Template
Use `[[../Standards/Task-Standard]]` e o template de exemplo dentro desse padrao.
