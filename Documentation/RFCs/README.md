# RFCs

Repositorio de decisoes tecnicas formais.

## Ciclo de vida
`Draft -> Review -> Accepted -> Superseded` ou `Rejected`

## Convencao
- ID: `RFC-YYYY-NNN`
- Nome sugerido: `RFC-YYYY-NNN - titulo-curto.md`

## Estrutura
- `Templates/RFC-Template.md`
- `Draft/`, `Review/`, `Accepted/`, `Rejected/`, `Superseded/`

## Vinculos obrigatorios
- RFC deve listar tasks derivadas (`TASK-*`)
- Task deve apontar para RFC mae quando aplicavel
