# Task Execution Checklist

Checklist padrao para qualquer agente antes de mover status da task.

## Antes de iniciar
- [ ] `task_id` informado no pedido
- [ ] `owner_role` compativel com o escopo
- [ ] Dependencias e bloqueios validados
- [ ] RFC vinculada quando aplicavel

## Durante execucao
- [ ] Atualizar secao de log na task
- [ ] Registrar decisoes relevantes
- [ ] Manter rastreabilidade de arquivos/artefatos

## Antes de concluir
- [ ] Criterios de aceite atendidos
- [ ] Hand-off descrito (se necessario)
- [ ] Status da task atualizado
