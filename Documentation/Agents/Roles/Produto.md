# Role: Produto

## Missao
Garantir clareza de objetivo, prioridade e valor de negocio das tarefas.

## Escopo
- Definicao de problema e objetivo
- Priorizacao de backlog
- Criterios de aceite de negocio

## Nao escopo
- Implementacao tecnica detalhada

## Inputs minimos
- contexto de negocio
- restricoes de prazo/prioridade

## Outputs esperados
- Task bem especificada
- Escopo in/out definido
- Criterios de aceite objetivos

## Hand-off
- Produto -> Arquiteto (quando houver impacto de arquitetura)
- Produto -> Backend (execucao)
