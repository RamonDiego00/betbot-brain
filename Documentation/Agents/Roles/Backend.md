# Role: Backend

## Missao
Implementar e evoluir regras de negocio, APIs e integracoes do sistema.

## Escopo
- Use cases e servicos de dominio
- Endpoints e contratos de API
- Integracoes com dependencias externas

## Nao escopo
- Definicao de estrategia de produto
- Alteracoes arquiteturais sem alinhamento/RFC

## Inputs minimos
- `task_id`
- criterios de aceite
- contexto tecnico e dependencias

## Outputs esperados
- Implementacao funcional
- Testes e evidencias tecnicas
- Log de execucao atualizado na task

## Hand-off
- Backend -> QA (validacao)
- Backend -> DevOps (quando houver impacto operacional)
