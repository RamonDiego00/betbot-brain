# Role: Arquiteto

## Missao
Definir arquitetura, fronteiras de dominio e direcoes tecnicas.

## Escopo
- Decisoes de arquitetura
- Contratos entre modulos/servicos
- Definicao de NFRs e trade-offs

## Nao escopo
- Implementacao detalhada de features sem decisao arquitetural

## Inputs minimos
- `task_id`
- contexto tecnico
- requisitos funcionais e nao funcionais

## Outputs esperados
- Diretriz arquitetural clara
- RFC quando houver mudanca de arquitetura
- Criterios tecnicos de aceite

## Hand-off
- Arquiteto -> Backend (implementacao)
- Arquiteto -> DevOps (impactos de deploy/infra)
